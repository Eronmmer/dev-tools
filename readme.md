# A List of awesome tools for Frontend Developers


## HTML

- [HTML5 Boilerplate](https://html5boilerplate.com/)

- [Email Boilerplate](https://github.com/seanpowell/Email-Boilerplate)

- [HTML5 Reference](http://htmlreference.io/)

- [Convert Markdown to HTML](https://daringfireball.net/projects/markdown/)

## CSS

- [Generate pretty looking waves in SVG format](https://getwaves.io)

- [A tool for making CSS clip paths](http://bennettfeely.com/clippy/)

- [CSS Reset](http://cssreset.com/)

- [Sass(CSS with superpowers)](https://sass-lang.com)

- [MDN CSS Reference](https://developer.mozilla.org/en-US/docs/Web/CSS/Reference)

- [Generate basic CSS Grid code to make dynamic layouts](https://cssgrid-generator.netlify.com/)

- [A simple CSS library that lets you add animations with ease](http://daneden.github.io/animate.css/)

- [Easing functions you can use in CSS transitions and animations](https://easings.net/en)

- [A collection of beautiful colour gradients](http://uigradients.com/)

## JavaScript

- [A resource to help figure out what JavaScript array method would be best to use at any given time](https://arrayexplorer.netlify.com/)

- [A resource to help figure out what JavaScript object method would be best to use at any given time](https://objectexplorer.netlify.com/)

- [An extensive math library for JavaScript and Node.js](https://mathjs.org)

- [A javascript library for formatting and manipulating numbers](http://numeraljs.com)

- [Babel(A JavaScript compiler)](https://babeljs.io)

- [A typed superset of JavaScript that compiles to plain JavaScript](https://www.typescriptlang.org)

- [A linting utility for JavaScript and JSX](https://eslint.org)

- [Node Package Manager for JavaScript](https://npmjs.org)

- [JavaScript Reference by MDN](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference)

- [JavaScript KeyCode Reference](http://keycodes.atjayjo.com/)
